# Real-Time Weather Dashboard

A small dashboard that fetches live weather data from WeatherAPI using modern asynchronous JavaScript.

## Features

- Fetches weather data with `fetch()` and `async/await`
- Uses the provided WeatherAPI key for real-time live weather
- Displays actual weather condition icons for sunny, cloudy, rainy, and stormy skies
- Shows temperature, feels-like, humidity, wind speed, and wind direction
- Search by city name and show Indian locations with a special Andhra Pradesh theme
- Colorful animated dashboard using sky, sun, rain, and wind imagery

## Run locally

1. Open `index.html` in a browser, or serve the folder with a local static server.
2. Enter a city name and click `Search`.

## Notes

This dashboard uses the free Open-Meteo geocoding service, so no API key is required.
